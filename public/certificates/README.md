# Certificate Templates

Place your certificate template images in this directory.

## Required Templates

- `template1.png` - Default certificate template (1200x850px recommended)

## Template Requirements

- **Format**: PNG or JPG
- **Dimensions**: 1200x850px (landscape) or 850x1200px (portrait)
- **Design**: Include background design, borders, logos, but leave space for dynamic text
- **Colors**: Professional colors matching UMak brand

## Adding New Templates

1. Save your template image in this directory
2. Name it descriptively (e.g., `certificate-appreciation.png`)
3. Update the template list in the application code

## Sample Data

For testing with dynamic text placeholders:
- {{name}} - Recipient's full name
- {{title}} - Recipient's job title or achievement
- {{date}} - Event or completion date
- {{email}} - Recipient's email address
